# 예산관리 엑셀 자동생성 스크립트

항목명/예산액/집행액을 넣으면 예산관리용 xlsx를 만들어주는 파이썬 스크립트.
집행률이 **80% 이상이면 노란색**, **100% 이상이면 빨간색**으로 행 전체가 칠해진다.

## 핵심: 살아있는 파일

잔액과 집행률을 파이썬이 계산해서 숫자로 박지 않고 **엑셀 수식**으로 넣고,
색상도 **조건부 서식**으로 넣는다. 그래서 파일을 만든 뒤 엑셀에서
집행액만 고쳐도 잔액·집행률·경고색이 전부 자동으로 따라온다.

| 열 | 내용 | 들어가는 값 |
|----|------|-------------|
| A | 항목명 | 문자열 |
| B | 예산액 | 숫자 |
| C | 집행액 | 숫자 (여기만 고치면 된다) |
| D | 잔액 | `=B2-C2` |
| E | 집행률 | `=IF(B2=0,0,C2/B2)` |

맨 아래 합계 행은 `SUBTOTAL`을 써서, 자동 필터로 일부 항목을 숨기면
보이는 항목만 합산된다.

## 설치

```bash
pip install -r requirements.txt
```

## 사용법

샘플 데이터로 테스트 파일 만들기:

```bash
python budget_excel.py
```

→ `sample_budget.xlsx` 생성

내 데이터로 만들기:

```python
from budget_excel import create_budget_file

items = [
    ("사무용품비", 1_000_000, 450_000),   # (항목명, 예산액, 집행액)
    ("교육훈련비", 2_000_000, 1_600_000),
]
create_budget_file(items, "2026년_예산.xlsx")
```

## 샘플 데이터

경계값이 전부 들어있어서 파일을 열면 규칙이 맞는지 눈으로 바로 확인된다.

| 항목명 | 예산액 | 집행액 | 집행률 | 색 |
|--------|--------|--------|--------|-----|
| 사무용품비 | 1,000,000 | 450,000 | 45.0% | 없음 |
| 교육훈련비 | 2,000,000 | 1,600,000 | 80.0% | 노랑 (경계) |
| 출장여비 | 3,000,000 | 2,760,000 | 92.0% | 노랑 |
| 회의비 | 500,000 | 500,000 | 100.0% | 빨강 (경계) |
| 소모품비 | 800,000 | 920,000 | 115.0% | 빨강 |
| 예비비 | 0 | 0 | 0.0% | 없음 |

"80% 넘으면"은 `>= 80%`로 해석했다. 정확히 80.0%인 교육훈련비도 노랑에 포함된다.
예산액이 0이어도 `IF`가 방어해서 `#DIV/0!` 대신 0%가 나온다.

## 테스트

```bash
python -m pytest test_budget_excel.py -v
```

테스트는 파일을 만든 뒤 openpyxl로 다시 읽어 수식·서식 규칙·숫자 형식을 확인한다.

**한계:** 조건부 서식의 실제 렌더링 색과 수식의 계산 결과는 엑셀이 파일을
열 때 계산하는 것이라, 스크립트로는 규칙과 수식이 올바르게 *기록됐는지*까지만
확인할 수 있다. 노란색이 실제로 화면에 보이는지는 파일을 열어서 봐야 한다.
